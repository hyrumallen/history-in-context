# Polish Pass Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Four focused polish changes — load Inter font, remove dead CSS, add event descriptions with native tooltips, and add a ↗ link indicator on events with Wikipedia links.

**Architecture:** All changes are isolated to individual files with no new components or dependencies. Events get a `description` field; `EventCell` reads it for the tooltip and renders a ↗ glyph when a link is present.

**Tech Stack:** React 18, Vite, plain CSS-in-JS (no styling library). No test suite — verification is visual in the browser (`npm run dev`).

## Global Constraints

- No new npm dependencies
- No changes to grid layout, row height, column widths, or data schema beyond adding `description` to events
- All 147 events must have a non-empty `description` string
- `↗` is a plain Unicode character (U+2197), not an SVG icon

### Task 1: Load Inter from Google Fonts

**Files:**
- Modify: `index.html`

**Interfaces:**
- Produces: Inter loaded via Google Fonts; `font-family: 'Inter'` in `index.css` resolves to the real typeface

- [ ] **Step 1: Add Google Fonts link tags to `index.html`**

Open `index.html`. Inside `<head>`, after the existing `<meta>` tags and before `<title>`, add:

```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;600;700&display=swap" rel="stylesheet">
```

The full `<head>` section should look like:

```html
<head>
  <meta charset="UTF-8" />
  <link rel="icon" type="image/svg+xml" href="/vite.svg" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <link rel="preconnect" href="https://fonts.googleapis.com">
  <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
  <link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;600;700&display=swap" rel="stylesheet">
  <title>History in Context</title>
</head>
```

- [ ] **Step 2: Verify in browser**

Run `npm run dev`, open `http://localhost:5173`. Open DevTools → Network → filter by "fonts.googleapis" — you should see the Inter stylesheet load. The text in the grid should render in Inter rather than the system fallback.

- [ ] **Step 3: Commit**

```bash
git add index.html
git commit -m "feat: load Inter from Google Fonts"
```
